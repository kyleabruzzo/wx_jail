# WX JAIL - Advanced jail system made using ox_lib

# Features
- OX Lib implementation (notifications, target, context menu, ...)
- Target support
- Exploit protection
- Optimized
- Large config file
- Multiple jobs support
- Multiple jail locations
- [WX Logs](https://github.com/nwvh/wx_logs) implementation
